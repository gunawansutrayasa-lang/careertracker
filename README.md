# 🚀 CareerTracker

CareerTracker is a web-based portfolio management system that helps users organize and manage their professional journey in one place. Users can showcase projects, certifications, work experiences, skills, achievements, and organizational activities through a clean and intuitive dashboard.

## ⚙️ Installation

Clone this repository

```bash
git clone https://github.com/yourusername/CareerTracker.git
```

Go to project directory

```bash
cd CareerTracker
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=careertracker
PORT=3000
```

Run the application

```bash
npm start
```

Open your browser

```
http://localhost:3000
```
